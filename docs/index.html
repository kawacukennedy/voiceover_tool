Offline Python TTS Voiceover Tool

This is a fully offline text-to-speech application for generating high-quality voiceovers.

Installation

pip install -r requirements.txt

Usage

CLI

python main.py synth --text "Hello world" --voice narrator --out output.mp3

GUI

python gui_main.py

Features

  • Multiple voices
  • SSML support
  • Batch processing
  • REST API

Troubleshooting

If you encounter issues, ensure all dependencies are installed and models are available.
